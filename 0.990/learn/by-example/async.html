    worker w1 {
        int n = wait f;
        int sq = square(n);
        sq -> w2;

    worker w2 returns int {
        int n = wait f;

Async

Async functionality in Ballerina is a way to execute operations in an asynchronous manner by returning a future. The future can be used to extract the result of the async function call (if available), check the running status of an execution, or cancel the operation if needed. You can wait for a single async function or multiple async functions to complete using the wait expression.

async action call

 
    future<int> f4 = start square(20);
    future<string> f5 = start greet("Bert");

    int|string anyResult = wait f4 | f5;
    io:println(anyResult);

You can wait for either of the asynchronous functions to finish. Here f4 will finish before f5 since runtim:sleep() is called in the greet() function to delay its execution. The value returned by the asynchronous function that finishes first will be taken as the result.

 
    future<int> f6 = start sum(40, 60);
    future<int> f7 = start cube(3);
    future<string> f8 = start greet("Moose");

    map <int|string> resultMap = wait {first_field: f6, second_field: f7, 
                                            third_field: f8};
    io:println(resultMap);

You can wait for all of the above asynchronous functions to finish. The result of all these functions can be assigned to a map or a record.

 
    future<string> f9 = start greet("Bert");
    record { int first_field; int second_field; string third_field; } rec =
                    wait {first_field: f6, second_field: f7, third_field: f9};
    io:println("first field of record --> " + rec.first_field);
    io:println("second field of record --> " + rec.second_field);
    io:println("third field of record --> " + rec.third_field);
}
