Ballerina To Swagger

The Ballerina to Swagger tool is used to export an Open API Specification(OAS) v3 definition of a Ballerina service. This tool accepts the following command line arguments, -s | --service If the provided file contains multiple services, define a specific service that needs to be exported. -o | --output The destination path to write the exported OAS definition.

import ballerina/http;
import ballerina/log;

listener http:Listener helloEp = new(9090);

service hello on helloEp {
    resource function hi(http:Caller caller, http:Request request) {
        http:Response res = new;
        res.setPayload("Hello World!");

        var result = caller->respond(res);
        if (result is error) {
           log:printError("Error when responding", err = result);
        }
    }
}

#Run the following command in the Ballerina tools distribution to export the OAS definition of a ballerina service.
