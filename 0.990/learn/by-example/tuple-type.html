Tuples

A tuple is an immutable list of two or more values of fixed length. A tuple type is described by specifying the type for each member in the list.

    (int, string) a = (10, "John");
    io:println(a);

Here, you can define the type of a as a pair that consists of an int and a string.

 
    int aint;
    string astr;
    (aint, astr) = a;
    io:println(aint);
    io:println(astr);

This has a tuple of variable names to the left, and a variable referent of which the type is a tuple. This assignment statement assigns values of the tuple in the right to the variables in the left. In Ballerina, this is referred to as tuple destructuring.

 
    var (aint1, astr1) = a;

You can declare and assign values with var as shown here.

 
    var (q, r) = divideBy10(6);
    io:println("06/10: " + "quotient=" + q + " " + "remainder=" + r);

This invokes a function that returns a tuple.

 
    var (q1, _) = divideBy10(57);
    io:println("57/10: " + "quotient=" + q1);

To ignore a return value, use ‘_‘. This ignores the second return value.

 
    var (_, r1) = divideBy10(9);
    io:println("09/10: " + "remainder=" + r1);
}

This ignores the first return value.

 
function divideBy10(int d) returns (int, int) {
    int q = d / 10;
    int r = d % 10;
    return (q, r);
}

This function returns a tuple of two integers.
