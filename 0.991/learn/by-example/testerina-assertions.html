Assertions

Testerina has in-built assertions that enable developers to assert an outcome against an expected outcome. This sample illustrates how to use different assertions.

    int answer = 0;
    int a = 5;
    int b = 3;
    answer = intAdd(a, b);
    test:assertEquals(answer, 8, msg = "int values not equal");
}

assertEquals() function allows you to compare primitive types (e.g., int) to composite objects. Compare values of type int.
