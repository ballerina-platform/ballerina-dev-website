import ballerina/io;
import ballerina/java;

// Define a Ballerina function which will act as a Java field getter.
public function pi() returns float = @java:FieldGet {
    name: "PI",
    class: "java/lang/Math"
} external;

public function main() {
    float r = 4;
    // If a field is an instance field, the receiver instance has to be provided as the first parameter.
    float l = 2 * pi() * r;
    io:println(l);
}

Access/Mutate Java Fields

Java fields (both static and instance) can be read and updated in Ballerina. From the Ballerina side, they will appear as function calls. This example shows how to read a field value. Similarly, the @java:FieldSet annotation can be used to create a Ballerina function with an external function that acts as a field setter.
