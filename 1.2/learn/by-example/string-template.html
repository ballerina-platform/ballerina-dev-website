import ballerina/io;

public function main() {
    string name = "Ballerina";
    // Create a `string` template embedding the `name` variable.
    string template = string `Hello ${name}!!!`;
    // Print the defined string value.
    io:println(template);
}

String Template Literal

String templates are string literals, which allow embedded expressions of simple basic types (except nil). The placeholders for expressions are indicated by the dollar symbol followed by open and close curly braces ${}. You can enclose a string literal with the keyword string followed by two back-tick characters.
