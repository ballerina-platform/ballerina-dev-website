import ballerina/io;

public function main() {
    // The variable type is inferred from the expression on the right-hand side.
    // This is the same as `int k = 5`;
    var k = 5;
    io:println(10 + k);

    // The type of `strVar` is `string`.
    var strVar = "Hello!";
    io:println(strVar);
}

Var

Ballerina allows you to declare variables using the var keyword instead of specifying a type. The type of the declared variable is inferred from the expression on the right-hand side.
