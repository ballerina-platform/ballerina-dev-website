Data Manipulation with Functional Programming and Queries in Ballerina

In his latest InfoQ article, Yehonathan Sharvit goes deeper into functional programming (FP) capabilities in Ballerina and explores an innovative way to use Ballerina’s query language to express data manipulation logic and discusses a data structure called “table”.

A Guide for Date/Time Conversions in Ballerina

Here's a guide by WSO2 Software Engineer, Sahan Hemachandra, that covers the string manipulations required for the conversion of time formats into Ballerina time representations.

Secure a Ballerina REST API with OpenID Connect using Asgardeo

In this tutorial, Associate Director - Developer Advocacy at WSO2, Dakshitha Ratnayake, shows how to secure a Ballerina REST API by configuring JSON Web Token authentication for OpenID Connect with Asgardeo as its identity provider.
