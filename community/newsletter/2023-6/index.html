Unveiling the best of Ballerina Swan Lake: A year-end recap

As we wrap up this amazing year, we invite you to celebrate the significant milestones and achievements that shaped the path of the Ballerina programming language in 2023!

Feature highlights

In 2023, Ballerina underwent several significant feature updates, cementing its position as a versatile and powerful tool for writing integrations. Some of these key enhancements include:

  • GraalVM Native Images: Official support for native image generation with certified Ballerina library packages.
  • ballerina/lang.regexp: First-class support for regular expressions in the language, allowing compile-time validation with better tooling for regexp.
  • Ballerina Central: Redesigned with a fresh look and feel optimized for a superior user experience.
  • Java 17: With Swan Lake Update 8, Ballerina now produces Java 17-compatible executables.
  • EDI: Working with EDI data in Ballerina is now as simple as working with JSON and XML data.
  • Healthcare: With its native support for healthcare standards like FHIR, HL7, and X12, Ballerina enables rapid health tech application development.
  • bal build: Package build times have been reduced by roughly 30%, and more improvements are in the works.
  • bal persist: Simplified data persistence in Ballerina, offering an intuitive way to model data and type-safe clients to work with disparate data stores.
  • Query expressions: Enhanced query expressions with the "collect" clause of aggregation and the "group by" clause for grouping.
  • VS Code extension updates: Introduced Entity Relationship Diagram View for persist model, GraphQL service designer, Config file generator support, and the capability to import XML as records. Additionally, various improvements were made to the data mapper functionality.

Keep an eye out for more exciting developments as we strive to elevate Ballerina's capabilities and user experience.

Use cases

Ballerina was primarily designed for integration and excels in streamlining and simplifying complex integration scenarios. The language enables developers to effortlessly create, deploy, and manage APIs, services, and data connectors.

In 2023, we showcased the language's versatility in handling diverse paradigms, data types, and specialized developer tools, highlighting its adaptability across multiple use cases. These include:

  • Integration
  • AI
  • Healthcare
  • Data-oriented programming
  • EDA
  • B2B
  • Microservices
  • BFF

Stay tuned as we continue to showcase the expanding landscape of possibilities that Ballerina offers.
