Ballerina Library Packages are GraalVM Certified!

Did you know that Ballerina library packages are officially GraalVM certified? This certification ensures that the Ballerina compiler can verify if your code is compatible with generating a GraalVM native image, enabling seamless native compilation for faster startup times and reduced memory usage. Visit Ballerina Central today to explore your favorite packages and see if they’re ready for GraalVM!
