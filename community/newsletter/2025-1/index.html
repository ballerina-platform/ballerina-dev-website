Latest Release: Ballerina Swan Lake Update 11

Ballerina Swan Lake Update 11 (2201.11.0) brings Java 21 support, new WSDL and XSD tools, enhanced package documentation, and new features for Ballerina libraries and developer tools. The Ballerina concurrency model has been reimplemented using Java Virtual Threads and the latest concurrency enhancements in Java 21, resulting in a 2x performance improvement for CPU-bound operations, reduced memory consumption, and optimized GraalVM execution time and executable size.

Check out the full release highlights and upgrade today!
