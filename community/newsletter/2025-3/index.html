Ballerina Swan Lake Update 13 (2201.13.0)

We are excited to announce the thirteenth update release of Ballerina Swan Lake (2201.13.0). This update introduces experimental natural expression capabilities with first-class syntax for working with LLMs, multi-package workspace support for managing complex projects, decoupled CLI tools for faster updates, and intelligent build caching for improved developer productivity. Read the release blog post by Asma Jabir, WSO2 Technical Lead.

Building Reliable and Replayable Message Pipelines in Ballerina

In today’s interconnected world, message-based systems are critical, and losing a message can be costly. This article by Krishnananthalingam Tharmigan explores how to build reliable and replayable message pipelines using the Ballerina programming language. It demonstrates a practical use case that reads messages over a network, processes them through multiple stages, and delivers them to several destinations with built-in failure handling and replay support.

Using an Inline AI Agent for Financial Fraud Detection

Financial fraud detection systems have long been deployed across the industry. In his recent blog post, Chintana Wilamuna examines how to integrate a fraud-detection engine using inline AI agents within an existing integration workflow, focusing on classifying potentially fraudulent transactions.

The Ultimate Idea Hackathon

The IEEE Student Branch of the University of Moratuwa, Sri Lanka organized The Ultimate Idea Hackathon, an inter-university hackathon in October 2025, held at the WSO2 Colombo office. The hackathon attracted around 2000 registrants and 500 teams, as well as 100+ open source projects.

Winners: SyntaxSquad led by Imasha Jayarathnenwith Anjana Edirisinghe and Dishma Gangulel, for their project Sri Lanka Transparent Governance Platform.
