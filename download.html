{{ site.data.swanlake-latest.metadata.display-version }}

If you are already using Ballerina 2201.0.0 (Swan Lake), run the command below to directly update to {{ site.data.swanlake-latest.metadata.display-version }} using the Ballerina Update Tool.


bal dist pull {{ site.data.swanlake-latest.metadata.version }}


If you are using a version below 2201.0.0 (Swan Lake), run the commands below to update to {{ site.data.swanlake-latest.metadata.display-version }}.


1. Run bal update to get the latest version of the update tool.

2. Run bal dist pull {{ site.data.swanlake-latest.metadata.version }} to update your Ballerina version to {{ site.data.swanlake-latest.metadata.display-version }}.


However, if you are using a version below 2201.0.0 (Swan Lake) and if you already ran bal dist update (or bal dist pull {{ site.data.swanlake-latest.metadata.version }}) before bal update, see Troubleshooting to recover your installation.

{{ site.data.stable-latest.metadata.version }}

If you already have jBallerina version 1.2.0 or above installed, you can use the Ballerina update tool to directly update to {{ site.data.stable-latest.metadata.version }}. To do this, first, execute the command below to get the update tool updated to its latest version. 

ballerina update

Next, execute the command below to update to jBallerina {{ site.data.stable-latest.metadata.version }}.

bal dist update

For further details, see the Release Note.
