## Overview of Ballerina Swan Lake Update 12 (2201.12.11)

<em>Swan Lake Update 12 (2201.12.11) is the tenth patch release of Ballerina 2201.12.0 (Swan Lake Update 12) and it includes a new set of bug fixes to the package resolution.</em>

## Update Ballerina

Run the command below to update your current Ballerina installation directly to 2201.12.11 by using the [Ballerina Update Tool](/learn/update-tool/).

```
$ bal dist pull 2201.12.11
```

## Install Ballerina

If you have not installed Ballerina, then, download the [installers](/downloads/#swanlake) to install.
