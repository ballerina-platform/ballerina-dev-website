## Overview of Ballerina Swan Lake Update 13 (2201.13.1)

<em>Swan Lake Update 13 (2201.13.1) is the first patch release of Ballerina 2201.13.0 (Swan Lake Update 13). This patch includes several workspace-related fixes that enhance the functionality of the Low Code view.</em>

## Update Ballerina

Run the command below to update your current Ballerina installation directly to 2201.13.1 by using the [Ballerina Update Tool](/learn/update-tool/).

```
$ bal dist pull 2201.13.1
```

## Install Ballerina

If you have not installed Ballerina, then, download the [installers](/downloads/#swanlake) to install.
