cancel

Requests cancellation of a future.

cancel

(future f)

Requests cancellation of a future. This sets the cancellation flag in the strand corresponding to f. Each time that a strand yields, it will check the cancellation flag and terminate abnormally if the flag is set.

Parameters

  • f future

  • the future to be cancelled
