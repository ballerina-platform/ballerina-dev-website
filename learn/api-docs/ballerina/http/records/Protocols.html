Record - http : Protocols

A record for configuring SSL/TLS protocol and version to be used.

Fields

  • name string

  • SSL Protocol to be used (e.g.: TLS1.2)

  • versions string[] (default [])

  • SSL/TLS protocols to be enabled (e.g.: TLSv1,TLSv1.1,TLSv1.2)
