Record - xmlutils : JsonOptions

Represents a record type to provide configurations for the JSON to XML conversion.

Fields

  • attributePrefix string (default @)

  • The attribute prefix to use in the XML representation

  • arrayEntryTag string (default root)

  • The XML tag to add an element from a JSON array
