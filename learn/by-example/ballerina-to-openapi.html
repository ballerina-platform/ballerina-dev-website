    resource function get hi(http:Caller caller, http:Request request) {
        http:Response res = new;
        res.setPayload("Hello World!");

        var result = caller->respond(res);
        if (result is error) {
           log:printError("Error when responding", 'error = result);
        }
    }
}

Ballerina To OpenAPI

The Ballerina to OpenApi tool is used to generate an OpenAPI Specification(OAS) v3 definition of a Ballerina service. This tool’s command structure is as follows. bal openapi [-i <ballerina-file-path>] [-o <openapi-contract-path>]

For more information, see Ballerina OpenAPI Tools.
