import ballerina/io;

public function main() {
    int m = 1;

    // Integer literals can be hexadecimal (but not octal).
    int n = 0xFFFF;

    // You can use compound assignment operations such as `+=` and `-=`. 
    n += m;
    io:println(n);
}

Integers

The int type is 64-bit signed integers (same as long in Java) and supports usuall arithmetic operators: + - / %. Operator precedenc is same as C. Integer overflow is a runtime error in Ballerina.
