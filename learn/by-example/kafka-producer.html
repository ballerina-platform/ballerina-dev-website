import ballerinax/kafka;

public type Order record {|
    int orderId;
    string productName;
    decimal price;
    boolean isValid;
|};

// Create a subtype of `kafka:AnydataProducerRecord`
public type OrderProducerRecord record {|
    *kafka:AnydataProducerRecord;
    Order value;
    int key;
|};

kafka:Producer orderProducer = check new (kafka:DEFAULT_URL);

public function main() returns error? {
    OrderProducerRecord producerRecord = {
        key: 1,
        topic: "test-kafka-topic",
        value: {
            orderId: 1,
            productName: "ABC",
            price: 27.5,
            isValid: true
        }
    };
    // Sends the message to the Kafka topic.
    check orderProducer->send(producerRecord);
}

Producer

This shows how to send messages to a Kafka topic using a kafka:Producer client. For this to work properly, an active Kafka broker should be present.

For more information on the underlying module, see the Kafka module.
