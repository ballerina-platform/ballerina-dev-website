Clients - ftp : Client

Represents an FTP client that intracts with an FTP server

Remote Methods

get

(string path)

returns ReadableByteChannel | error
Isolated Function

The get() function can be used to retrieve file content from a remote resource.

 io:ReadableByteChannel|error channel = client->get(path);

Parameters

  • path string

  • The resource path

  • Return Type

    (ReadableByteChannel | error)

  • A ReadableByteChannel that represents the data source to the resource or an error if failed to establish communication with the FTP server or read the resource

(string path, ReadableByteChannel | string | xml | json content)

returns error?
Isolated Function

The append() function can be used to append content to an existing file in an FTP server.

 error? response = client->append(path, channel);

Parameters

  • The resource path

  • content ReadableByteChannel | string | xml | json

  • Content to be written to the file in server

  • Return Type

    (error?)

  • An error if failed to establish communication with the FTP server

returns error?

Parameters

  • path string

  • The resource path

  • Content to be written to the file in server

  • compressInput boolean (default false)

  • True if file should be compressed before uploading
