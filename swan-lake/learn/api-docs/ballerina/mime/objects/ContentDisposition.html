Object - mime : ContentDisposition

Represents values in Content-Disposition header.

Methods

toString

Converts the ContentDisposition type to a string suitable to use as the value of a corresponding MIME header.

Fields

  • fileName string (default "")

  • Default filename for storing the body part if the receiving agent wishes to store it in an external file

  • disposition string (default "")

  • Indicates how the body part should be presented (inline, attachment, or as form-data)

  • name string (default "")

  • Represents the field name in case of multipart/form-data

  • parameters map<string> (default {})

  • A set of parameters specified in the attribute=value notation
