Record - mysql : SSLConfig

SSL Configuration to be used when connecting to mysql server.

Fields

  • mode SSLMode (default SSL_PREFERRED)

  • SSLMode to be used during the connection

  • clientCertKeystore KeyStore?

  • Keystore configuration of the client certificates

  • trustCertKeystore KeyStore?

  • Keystore configuration of the trust certificates
