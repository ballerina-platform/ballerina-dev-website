Annotations - websub

  • SpecificSubscriber service

  • Annotation to declare that the service represents a specific webhook. Generic WebSub Subscriber service validation is not done for service variables annotated as @websub:SpecificSubscriber.

  • WebSub Subscriber Configuration for the service, indicating subscription related parameters.
