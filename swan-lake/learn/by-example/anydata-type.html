Anydata Type

The anydata type consists of plain data (a simple value, sequence value, or structured value that does not contain behavioral members at any depth). Thus, the anydata type is equivalent to ()|boolean|int|float|decimal|string|xml|anydata[]|map<anydata>|table<map<anydata>>. Although json and byte are not explicitly mentioned, they are subsets of the aforementioned union. The same is true for a record of which the type of each individual field is a subtype of anydata(including the rest field in open records).

Any Type

 
import ballerina/io;

public function main() {

    anydata data1 = 5;
    io:println(data1);

In this example, the variable named data1 of the anydata type holds an int value.

 
    int intVal = <int> data1;
    io:println(intVal + 10);

Before using the value of data1 in arithmetic operations, it is required to ascertain that is actually an int. A type cast or a type guard can be used for this.

 
    if data1 is int {
        io:println(data1 + 20);
    }

    int[] intArray = [1, 3, 5, 6];
    anydata dataArray = intArray;
    io:println(dataArray);
}

A variable of type anydata can hold any value of an anydata compatible type.
