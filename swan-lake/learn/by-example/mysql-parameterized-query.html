Parameterized Query

This example demonstrates how to use the sql:ParameterizedQuery to easily create pre-compiled SQL statements that can be reused. Note that the MySQL database driver JAR should be placed in theBALLERINA_HOME/bre/lib directory. The MySQL connector uses database properties from MySQL version 8.0.x onwards. Therefore, it is recommended to use a MySQL driver version greater than 8.0.x.

For more information on the underlying module, see the MySQL module.

returns sql:Error? {

    sql:ExecutionResult result =
        check mysqlClient->execute("DROP TABLE IF EXISTS Customers");
    io:println("Drop table executed: ", result);

Execute dropping the table. The sql:ExecutionResult is returned upon successful execution. An error will be returned in case of a failure.

Similarly, to drop a table, the create table query is executed. Here, the customerId is an auto-generated column.
