XML From Query Expression

Having the xml type as the target type for a query expression allows users to create an XML from that query expression. The resulted xml is the concatenation of the emitted values from the select clause. Therefore, every emitted value from the select clause must be of the type xml.

The xml element with nested children.

 
    xml authors = from var x in school/<student>/<firstName>

The from clause works similarly to a foreach statement. The authors is the concatenated xml of the query expression results.

 
        select <xml> x

The select clause is evaluated for each iteration. The emitted values are concatenated to form the xml result.

 
        limit 2;

The limit clause limits the number of output items.

 
    io:println(authors);
}
