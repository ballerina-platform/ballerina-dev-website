Send and Receive Emails

The Email Connector is used to send (with SMTP) and receive (with POP3 or IMAP4) emails using the SSL or STARTTLS protocols. This sample includes sending and receiving emails with default configurations over SSL using default ports.

For more information on the underlying module, see the Email module.

Defines the protocol specific configuration for the email listener. It can either be email:PopConfig for POP or email:ImapConfig for IMAP.

 
listener email:Listener emailListener = new ({
    host: "pop.email.com",
    username: "reader@email.com",
    password: "pass456",
    protocol: "POP",
    protocolConfig: popConfig,
    pollingInterval: 2000
});

Create the listener with the connection parameters and protocol related configuration. Polling interval specifies the time duration between each poll performed by the listener.

 
service emailObserver on emailListener {

One or many services can listen to the email listener for the periodically polled emails.

 
    resource function onMessage(email:Email emailMessage) {
        io:println("Email Subject: ", emailMessage.subject);
        io:println("Email Body: ", emailMessage.body);
    }

When an email is successfully received onMessage method is called

 
    resource function onError(email:Error emailError) {
        io:println("Error while polling for the emails: "
            + emailError.message());
    }

When an error occurs during the email poll operations onError is called

 
}
