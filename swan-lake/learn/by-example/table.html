Table

A table is a structural value whose members are mapping values that represent rows of the table. A table provides access to its members using a key that comes from the read-only fields of the member. It keeps its members in order but does not provide random access to a member using its position in this order. The built-in functions enable inserting, accessing, deleting data, and applying functions on members of a table. For more information on the underlying module, see the lang.table module.

Union Types

 
import ballerina/io;

type Person record {|
    readonly int id;
    string name;
    int age;
|};

type Employee record {
    readonly int id;
    string name;
    float salary;
};

This is the type created to represent a data row. The type of the value of the ID field is readonly and the field itself cannot be mutated. The ID is the unique identifier of an Employee.

 
type EmployeeTable table<Employee> key(id);

This is a table type. The EmployeeTable type has members that are employees, and within the EmployeeTable, each member is uniquely identified by its ID field.

 
type CustomerTable table<map<any>>;

You can define a table value whose members are of type map constrained by any type.

 
public function main() {

    EmployeeTable employeeTab = table [
      {id: 1, name: "John", salary: 300.50},
      {id: 2, name: "Bella", salary: 500.50},
      {id: 3, name: "Peter", salary: 750.0}
    ];

A table constructor works like a list constructor. The table preserves the order of its members. Iterating over the table gives the members of the table sequentially.

 
    io:println("Employee Table Information: ", employeeTab);

Prints the table data.

 
    io:println("Total number of Employees: ", employeeTab.length());

Returns the number of members of a table.

 
    Employee emp = {id: 4, name: "Max", salary: 900.0};
    employeeTab.add(emp);

Adds a new member to the Employee table.

 
    io:println("New Employee: ", employeeTab[4]);

Retrieves a member using the ID field of an Employee.

 
    io:println("Employee 1: ", employeeTab.get(1));

Returns the member of a table with a given key.

 
    io:println("Information of the removed Employee: ", employeeTab.remove(2));

Removes the member of a table with a given key and returns it.

 
    var personTblKeys = employeeTab.keys();
    io:println("Employee table keys: ", personTblKeys);

Gives a list of all keys of a table.

 
    Employee[] tableToList = employeeTab.toArray();
    io:println("Employee table to list: ", tableToList);

The members of a table can be returned as an array.

 
    string filtered = "";
    employeeTab.forEach(function (Employee employee) {
           if (employee.salary < 400.0) {
              filtered += employee.name;
              filtered += " ";
           }
    });
    io:println("Employees with salary < 400.0: ", filtered);

Tables are an iterable type and tables support functional iteration operations such as .forEach(), .map(), .filter(), and .reduce().

 
    table<Person> personTab = employeeTab.'map(function (Employee employee)
                                                            returns Person {
        return {id: employee.id, name:employee.name, age:23};
    });
    io:println("Person Table Information: ", personTab);

.map() applies a function to each member of a table and returns a table of the result. The resulting table will have the same keys as the argument table.
