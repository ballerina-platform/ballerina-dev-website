import ballerina/io;
import ballerina/task;
task:TimerConfiguration timerConfiguration = {
    intervalInMillis: 1000,
    initialDelayInMillis: 3000,
    noOfRecurrences: 10
};
listener task:Listener timer = new (timerConfiguration);int count = 0;
service timerService on timer {
    resource function onTrigger() {
        count += 1;
        io:println("MyCounter: ", count);
    }}

Task Service Timer

A task timer service can be used to execute tasks periodically. The task listener can be provided with a timer configuration that should have an intervalInMillis field specified and an initialDelayInMillis field if needed. The delay specifies the initial delay before the task is executed for the first time. Thereafter, the timer triggers at the provided interval. The onTrigger() resource is called when the timer triggers. An optional noOfRecurrences field can be used to provide the maximum number of times the timer should run before shutting down.

For more information on the underlying module, see the Task module.

import ballerina/io;

    intervalInMillis: 1000,
    initialDelayInMillis: 3000,

The task:TimerConfiguration record to configure the task listener.

 
    noOfRecurrences: 10
};

Number of recurrences will limit the number of times the timer runs.

 
listener task:Listener timer = new (timerConfiguration);

Initializes the listener using the configurations defined above.

 
int count = 0;

service timerService on timer {

Creating a service bound to the task listener.

        io:println("MyCounter: ", count);
    }

This resource triggers when the timer goes off.
