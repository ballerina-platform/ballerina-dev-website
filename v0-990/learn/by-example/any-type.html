Any Type

The any type is the root of Ballerina types. It can represent a variable of any type, excluding the error type. When you do not have prior knowledge of the type of a variable, you can assign it to a variable of type any. Values of these variables can come from dynamic content such as the request and response message references and Ballerina reference variable types. The any type allows you to skip compile-time type checks.
