import ballerina/io;
function printValue(string value) {
    io:println(value);
}
function add(int a, int b) returns (int) {
    return a + b;
}
public function main() {
    printValue("This is a sample text");
    int result = add(5, 6);
    io:println(result);
}

Functions

Functions operate the same way as in any other language. It is a mechanism to create a reusable unit of functionality within a program.
