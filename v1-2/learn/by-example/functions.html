import ballerina/io;

function printValue(string value) {
    io:println(value);
}

This function takes a string argument. However, it does not return a value.

 
function add(int a, int b) returns int {
    return a + b;
}

This function takes in two int values as arguments and returns their sum as an integer.

Ballerina does not allow ignoring the return value of a function invocation implicitly unless the return type is (). A return value can be ignored using _ if the value may never be an error (i.e., the return type of the function does not contain error).
