Tuple Destructure Binding Pattern

Tuple binding patterns are used to destructure and assign values to each individual variable of the tuple. The tuple destructuring binding patterns will be used to refer to existing variables as a tuple, destructure the given value on the right hand side, and assign the values to each individual variable of the tuple at runtime.

import ballerina/io;

public function main() {

    [string, [int, boolean]] [a1, [a2, a3]] = ["Hello", [123, true]];
    [a1, [a2, a3]] = ["Ballerina", [453, false]];
    io:println("Tuple variable : ", a1, " ", a2, " ", a3);

Tuple variables can be destructured with new values when used as a tuple variable reference based binding pattern.

 
    [string, [int, boolean]] [b1, [b2, b3]] = ["Hello", [123, true]];
    b1 = "Ballerina";
    b2 = 453;
    b3 = false;
    io:println("Tuple variable : ", b1, " ", b2, " ", b3);

The variable can also be updated individually because once they are defined they are treated as individual variables at runtime.

 
    [string, [int, [boolean, float]]] [c1, [c2, [c3, c4]]] =
                                        ["Test", [123, [true, 5.6]]];
    [[c1, c2], c3] = [["Ballerina", 453], false];
    [c2, c3, c4] = [657, true, 76.8];
    io:println("Tuple variable : ", c1, " ", c2, " ", c3, " ", c4);

The binding patterns are recursive in nature. These examples show how to write complex recursive variable references for destructuring.

 
    [[string, [int, [boolean, int]]], [float, int]] v1 =
                            [["Ballerina", [3, [true, 34]]], [5.6, 45]];
    [[string, [int, [boolean, int]]],
                        [float, int]] [[d1, [d2, [d3, d4]]], [d5, d6]] = v1;
    [[d1, [d2, [d3, d4]]], [d5, d6]] = bar();
    io:println("Tuple variable : ",
                    d1, " ", d2, " ", d3, " ", d4, " ", d5, " ", d6);

Tuple variables can also be referenced using tuple-type expressions.

 
    var [e1, [e2, e3]] = foo();
    io:println("Tuple variable : ", e1, " ", e2, " ", e3);

Tuple type can also take “var” as the type label in which the type will be inferred from the right hand side.

 
    var [[f1, [f2, [f3, f4]]], [f5, f6]] = bar();
    io:println("Tuple variable : ",
                            f1, " ", f2, " ", f3, " ", f4, " ", f5, " ", f6);
}

    return ["Ballerina", [453, false]];
}

function bar() returns [[string, [int, [boolean, int]]], [float, int]] {
    return [["Ballerina", [3, [true, 34]]], [5.6, 45]];
}
